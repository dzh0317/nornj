## Quick Start

### 1. Installation

```bash
npm install babel-plugin-nornj-in-jsx
```

### 2. Configure Babel

```js
{
  "plugins": [
    "nornj-in-jsx"
  ]
}
```

### 3. Getting Started

```js
const App = props => (
  <if condition={props.success}>
    <i>success</i>
    <else>
      <i>fail</i>
    </else>
  </if>
);

ReactDOM.render(<App success={false} />, document.querySelector('#app'));

//Render resultï¼š<i>fail</i>
```
